# TODO计划目录说明

本目录用于维护项目的开发计划与任务清单，支持按版本与里程碑进行管理与追踪。

## 内容结构
- 《01-开发计划》：版本路线、里程碑、风险与度量。
- 《02-项目TODO清单》：Backlog/In Progress/Done 持续更新。

## 维护规则
- 每周例会同步 TODO 状态与新增事项。
- 完成的事项进入版本发布说明并标注影响范围。
- 重大风险与变更需在开发计划中记录与评审。
