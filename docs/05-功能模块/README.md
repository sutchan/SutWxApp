# 功能模块目录说明

本目录包含 SutWxApp 的核心业务模块文档，覆盖用户、内容、电商、支付、积分、会员、活动、消息等。用于指导开发、测试与运维协作。

## 快速导航
- 《01-积分系统功能说明》：积分获取/消耗/规则/接口与交互。
- 《02-功能模块说明文档》：模块总览、依赖关系、数据流与版本规划。

## 文档约定
- 统一使用 UTF-8 无 BOM、Unix LF。
- 路径与引用以 `docs/` 为根，遵循中文命名及清晰层级。
- 详细接口请参考：`docs/03-开发者指南/03-API接口文档.md`。

## 贡献与维护
- 新增/变更模块需同时更新本目录 README 与相关文档。
- 保持术语一致性、接口统一性与规则可追溯性。
- 建议在变更时附带影响分析与测试要点。
