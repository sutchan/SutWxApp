## 2. 功能模块列表

### 2.1 积分系统

**功能描述**：通过购物、签到等方式获取积分，积分可用于抵扣现金

**使用场景**：
- 购物获取积分
- 签到获取积分
- 积分抵扣现金
- 查看积分明细

**功能特点**：
- 多种积分获取方式
- 积分使用灵活
- 积分明细清晰
- 积分有效期长
