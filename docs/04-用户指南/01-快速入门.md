## 1. 概述

### 1.1 文档目的

本文档旨在介绍如何快速开始使用SutWxApp，帮助新用户快速上手。

### 1.2 适用对象

- 新用户
- 首次使用SutWxApp的用户

## 2. 准备工作

### 2.1 硬件要求

- 智能手机（支持微信小程序）
- 稳定的网络连接

### 2.2 软件要求

- 微信6.7.2或更高版本

## 3. 安装和注册

### 3.1 安装微信小程序

1. 打开微信，点击底部的「发现」选项
2. 点击「小程序」选项
3. 在搜索框中输入「SutWxApp」
4. 点击搜索结果中的「SutWxApp」小程序
5. 点击「允许」按钮，授权小程序使用相关权限

### 3.2 注册账号

1. 进入SutWxApp小程序
2. 点击「我的」选项
3. 点击「登录/注册」按钮
4. 选择「微信登录」
5. 点击「允许」按钮，授权微信登录
6. 完成注册，进入个人中心

## 4. 首次使用

### 4.1 浏览商品

1. 进入SutWxApp小程序
2. 点击「首页」选项
3. 浏览推荐商品
4. 点击商品卡片，查看商品详情
5. 点击「加入购物车」按钮，将商品加入购物车

### 4.2 购物车管理

1. 点击底部的「购物车」选项
2. 查看购物车中的商品
3. 调整商品数量
4. 勾选要购买的商品
5. 点击「结算」按钮，进入订单确认页面

### 4.3 创建订单

1. 在订单确认页面，选择收货地址
2. 选择支付方式
3. 点击「提交订单」按钮
4. 进入支付页面，完成支付
5. 订单创建成功，进入订单详情页面

### 4.4 查看订单

1. 点击底部的「我的」选项
2. 点击「我的订单」选项
3. 查看所有订单
4. 点击订单，查看订单详情
5. 跟踪订单状态

## 5. 常见操作

### 5.1 修改个人信息

1. 点击底部的「我的」选项
2. 点击「个人信息」选项
3. 修改个人信息，包括昵称、头像等
4. 点击「保存」按钮，保存修改

### 5.2 管理收货地址

1. 点击底部的「我的」选项
2. 点击「收货地址」选项
3. 点击「新增地址」按钮，添加新的收货地址
4. 点击现有地址，修改或删除地址

### 5.3 查看积分

1. 点击底部的「我的」选项
2. 查看积分余额
3. 点击「积分明细」选项，查看积分明细

### 5.4 联系客服

1. 点击底部的「我的」选项
2. 点击「联系客服」选项
3. 进入客服聊天页面，发送消息

## 6. 快速导航

| 功能 | 入口 |
|------|------|
| 浏览商品 | 首页 |
| 搜索商品 | 首页顶部搜索框 |
| 购物车 | 底部导航栏「购物车」 |
| 个人中心 | 底部导航栏「我的」 |
| 订单管理 | 个人中心「我的订单」 |
| 收货地址 | 个人中心「收货地址」 |
| 积分管理 | 个人中心「积分明细」 |
| 联系客服 | 个人中心「联系客服」 |

## 7. 注意事项

1. 确保微信版本在6.7.2或更高版本
2. 确保手机网络连接稳定
3. 授权小程序使用相关权限，以便正常使用所有功能
4. 保护好个人信息，不要泄露账号密码
5. 如有问题，及时联系客服
