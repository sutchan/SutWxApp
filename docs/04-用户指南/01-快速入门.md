# SutWxApp 快速入门
## 1. 小程序介绍
**SutWxApp** 是一款功能丰富的微信小程序应用，致力于为用户提供便捷、高效的服务体验。通过本快速入门指南，您将了解如何快速开始使用我们的小程序。
## 2. 如何找到并打开小程序
### 2.1 通过微信搜索

1. 打开微信应用
2. 点击顶部搜索框
3. 输入 "SutWxApp" 或"苏特"（应用名称）
4. 在搜索结果中找到我们的小程序并点击进入
### 2.2 通过扫描二维码
1. 打开微信应用
2. 点击右上角"+" 按钮
3. 选择 "扫一扫"
4. 扫描以下二维码即可进入小程序

（此处预留二维码位置）
### 2.3 通过分享链接

如果您收到朋友分享的小程序链接或卡片，只需点击链接或卡片即可直接进入。
## 3. 首次使用指南

### 3.1 微信授权登录
首次使用小程序时，您需要进行微信授权登录：

1. 进入小程序后，系统会提示您进行授权
2. 点击 "允许" 按钮，授权微信账号登录
3. 可选授权：位置信息（用于提供本地服务）、通知权限（接收订单和活动通知）
### 3.2 完善个人信息

授权成功后，建议您前往"我的"页面，点击头像或昵称区域进入个人资料页，完善您的收货地址、联系方式等信息，以便后续购物和接收服务通知。

## 4. 核心功能快速了解
### 4.1 浏览与搜索商品
1. 在首页浏览推荐商品
2. 通过分类页面查看不同类别的商品
3. 使用搜索框查找特定商品
4. 点击商品图片或名称查看详情
### 4.2 购物车操作
1. 在商品详情页点击 "加入购物车"
2. 点击右上角购物车图标查看购物车内容
3. 在购物车中修改商品数量或删除商品
4. 勾选要购买的商品，点击 "结算"

### 4.3 下单与支付
1. 在结算页面确认收货地址和商品信息
2. 选择支付方式（微信支付）
3. 点击 "提交订单" 按钮
4. 完成微信支付流程

### 4.4 订单管理

1. 在"我的" 页面点击 "我的订单"
2. 查看不同状态的订单（待付款、待发货、待收货、待评价、已完成）
3. 点击订单详情查看更多信息
4. 执行相应操作（付款、确认收货、评价等）
## 5. 个性化设置

### 5.1 通知设置

1. 进入 "我的" 页面
2. 点击 "设置" 图标
3. 在设置页面管理通知偏好

### 5.2 地址管理

1. 进入 "我的" 页面
2. 点击 "地址管理"
3. 添加、编辑或删除收货地址
4. 设置默认收货地址

## 6. 获取帮助

如果您在使用过程中遇到任何问题：

1. 查看 [常见问题解答](./04-常见问题.md)
2. 在"我的" 页面点击 "联系客服"
3. 发送邮件至 support@sutwxapp.com

## 7. 更新与反馈
我们会定期更新小程序，提供更好的用户体验和新功能。请保持微信应用更新到最新版本，以确保小程序正常运行。
如果有任何建议或反馈，欢迎通过客服渠道或小程序内的反馈功能告诉我们！
