## 1. 概述

### 1.1 文档目的

本文档旨在详细介绍SutWxApp的使用方法，帮助用户全面了解和使用SutWxApp的所有功能。

### 1.2 适用对象

- 所有使用SutWxApp的用户

## 2. 首页

### 2.1 功能介绍

首页是SutWxApp的主要入口，包含以下功能：

- **轮播图**：展示热门活动和推荐商品
- **分类导航**：快速进入商品分类
- **推荐商品**：推荐热门商品
- **活动专区**：展示当前活动
- **搜索功能**：搜索商品

### 2.2 使用方法

1. **浏览商品**：滑动屏幕浏览推荐商品
2. **查看分类**：点击分类图标，进入商品分类页面
3. **搜索商品**：点击搜索框，输入关键词搜索商品
4. **查看活动**：点击轮播图或活动专区，查看活动详情

## 3. 分类页

### 3.1 功能介绍

分类页展示商品分类，包含以下功能：

- **分类列表**：展示所有商品分类
- **分类商品**：展示当前分类下的商品
- **筛选功能**：根据价格、销量等筛选商品
- **排序功能**：根据价格、销量等排序商品

### 3.2 使用方法

1. **选择分类**：点击分类列表中的分类，查看该分类下的商品
2. **筛选商品**：点击筛选按钮，选择筛选条件
3. **排序商品**：点击排序按钮，选择排序方式
4. **查看商品**：点击商品卡片，查看商品详情

## 4. 商品详情页

### 4.1 功能介绍

商品详情页展示商品的详细信息，包含以下功能：

- **商品图片**：展示商品图片
- **商品信息**：展示商品名称、价格、销量等
- **商品描述**：详细描述商品
- **规格选择**：选择商品规格
- **数量调整**：调整购买数量
- **加入购物车**：将商品加入购物车
- **立即购买**：直接购买商品
- **评价查看**：查看商品评价

### 4.2 使用方法

1. **查看商品图片**：滑动屏幕查看商品图片
2. **选择规格**：点击规格选项，选择商品规格
3. **调整数量**：点击「+」或「-」按钮，调整购买数量
4. **加入购物车**：点击「加入购物车」按钮，将商品加入购物车
5. **立即购买**：点击「立即购买」按钮，直接购买商品
6. **查看评价**：滑动屏幕到底部，查看商品评价

## 5. 购物车

### 5.1 功能介绍

购物车用于管理待购买的商品，包含以下功能：

- **商品列表**：展示购物车中的商品
- **数量调整**：调整商品数量
- **商品删除**：删除购物车中的商品
- **全选/取消全选**：全选或取消全选购物车中的商品
- **价格计算**：自动计算商品总价
- **结算功能**：结算购物车中的商品

### 5.2 使用方法

1. **查看商品**：查看购物车中的商品
2. **调整数量**：点击「+」或「-」按钮，调整商品数量
3. **删除商品**：点击商品右侧的删除按钮，删除商品
4. **全选商品**：点击顶部的全选按钮，全选购物车中的商品
5. **结算商品**：点击底部的「结算」按钮，进入订单确认页面

## 6. 订单确认页

### 6.1 功能介绍

订单确认页用于确认订单信息，包含以下功能：

- **收货地址**：选择或添加收货地址
- **商品列表**：展示待购买的商品
- **价格明细**：展示商品总价、运费、优惠等
- **支付方式**：选择支付方式
- **订单备注**：添加订单备注
- **提交订单**：提交订单

### 6.2 使用方法

1. **选择收货地址**：点击收货地址，选择或添加收货地址
2. **查看商品**：查看待购买的商品
3. **查看价格明细**：查看商品总价、运费、优惠等
4. **选择支付方式**：选择支付方式，如微信支付
5. **添加备注**：点击订单备注，添加订单备注
6. **提交订单**：点击「提交订单」按钮，进入支付页面

## 7. 支付页

### 7.1 功能介绍

支付页用于完成订单支付，包含以下功能：

- **订单信息**：展示订单信息
- **支付金额**：展示支付金额
- **支付方式**：展示选择的支付方式
- **支付按钮**：点击完成支付

### 7.2 使用方法

1. **确认订单信息**：确认订单信息和支付金额
2. **完成支付**：点击「立即支付」按钮，完成支付
3. **查看订单**：支付成功后，点击「查看订单」按钮，查看订单详情

## 15. 常见问题

### 15.1 如何修改密码？

1. 进入个人中心
2. 点击「设置」选项
3. 点击「账号安全」选项
4. 点击「修改密码」选项
5. 输入原密码和新密码
6. 点击「保存」按钮

### 15.2 如何查看订单物流？

1. 进入个人中心
2. 点击「我的订单」选项
3. 找到要查看的订单，点击「查看物流」按钮
4. 查看物流信息

### 15.3 如何申请退款？

1. 进入个人中心
2. 点击「我的订单」选项
3. 找到要退款的订单，点击「申请退款」按钮
4. 填写退款原因和说明
5. 点击「提交」按钮
6. 等待审核
