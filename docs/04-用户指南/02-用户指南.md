# SutWxApp 用户指南

## 1. 小程序介绍

**SutWxApp** 是一款为用户提供便捷服务的微信小程序应用。本用户指南将详细介绍小程序的各项功能、使用方法和技巧，帮助您充分利用我们的服务。

## 2. 界面导航

### 2.1 底部导航栏

小程序底部导航栏包含四个主要入口：

- **首页**：展示推荐商品、活动信息和快捷入口
- **分类**：商品分类浏览
- **购物车**：管理待购买商品
- **我的**：个人中心，包含订单、设置等功能

### 2.2 顶部导航栏

顶部导航栏通常包含：

- **搜索框**：搜索商品
- **消息图标**：查看通知和消息
- **购物车图标**：快速进入购物车

## 3. 用户账号管理

### 3.1 账号登录

1. 首次打开小程序时，系统会自动提示微信授权
2. 点击"允许"即可完成登录
3. 如果授权过期，可以在"我的"页面重新授权

### 3.2 个人资料管理

1. 进入"我的"页面
2. 点击头像或昵称区域进入个人资料设置
3. 可以修改昵称、头像（默认使用微信头像）
4. 绑定或更换手机号
5. 设置性别、生日等个人信息

### 3.3 账号安全

1. 在"设置"中可以查看账号绑定情况
2. 定期更新密码（如需独立密码登录）
3. 管理第三方授权绑定

## 4. 商品浏览与搜索

### 4.1 首页浏览

- **轮播图**：展示最新活动和推荐商品
- **分类入口**：快速进入热门分类
- **推荐商品**：根据您的偏好推荐相关商品
- **限时活动**：查看当前进行的限时促销

### 4.2 分类浏览

1. 点击底部导航栏"分类"进入分类页面
2. 选择感兴趣的商品分类
3. 可以按子分类进一步筛选
4. 使用排序功能（销量、价格、新品等）

### 4.3 搜索功能

1. 点击顶部搜索框
2. 输入关键词进行搜索
3. 查看搜索历史和热门搜索词
4. 使用搜索结果筛选（价格区间、品牌、属性等）

### 4.4 商品详情页

商品详情页包含以下信息和功能：

- 商品图片轮播
- 商品名称、价格、销量
- 商品详细描述
- 规格参数
- 用户评价
- 加入购物车按钮
- 立即购买按钮
- 收藏按钮
- 分享功能

## 5. 购物车管理

### 5.1 添加商品

1. 在商品详情页选择规格和数量
2. 点击"加入购物车"按钮
3. 可以选择继续购物或查看购物车

### 5.2 购物车操作

- **选择商品**：勾选要结算的商品
- **修改数量**：调整商品购买数量
- **删除商品**：移除不需要的商品
- **收藏商品**：将商品添加到收藏夹
- **清空购物车**：一键清除所有商品

### 5.3 购物车优惠

- 自动计算商品总价
- 显示可用优惠券
- 显示会员折扣
- 计算运费

## 6. 订单管理

### 6.1 创建订单

1. 在购物车中选择商品并点击"结算"
2. 确认收货地址（可修改或添加新地址）
3. 选择配送方式
4. 选择支付方式（微信支付）
5. 应用优惠券或积分
6. 确认订单信息并提交

### 6.2 订单状态

订单有以下几种状态：

- **待付款**：订单已创建，等待支付
- **待发货**：支付成功，等待商家发货
- **待收货**：商品已发出，等待收货
- **待评价**：已收到商品，等待评价
- **已完成**：订单已完成
- **已取消**：订单已取消

### 6.3 订单操作

根据订单状态，您可以进行以下操作：

- **待付款**：立即支付、取消订单
- **待发货**：查看详情、提醒发货
- **待收货**：查看物流、确认收货
- **待评价**：评价商品、申请售后
- **已完成**：再次购买、申请售后、查看评价

## 7. 支付功能

### 7.1 微信支付

1. 在订单确认页选择微信支付
2. 点击"提交订单"后，会跳转至微信支付界面
3. 输入支付密码或使用生物识别完成支付
4. 支付成功后会自动跳转回小程序

### 7.2 支付安全

- 所有支付信息通过微信安全通道传输
- 建议开启微信支付密码或生物识别
- 定期检查支付记录

## 8. 地址管理

### 8.1 添加新地址

1. 进入"我的"页面，点击"地址管理"
2. 点击"新增收货地址"
3. 填写收货人姓名、手机号、所在地区和详细地址
4. 可以设置为默认地址
5. 点击"保存"完成添加

### 8.2 编辑/删除地址

- **编辑**：点击地址进入编辑页面，修改后保存
- **删除**：在地址列表滑动删除或点击编辑页面的删除按钮
- **设为默认**：在地址列表设置默认收货地址

## 9. 优惠券与积分

### 9.1 优惠券使用

- 在"我的"页面查看可用优惠券
- 下单时自动显示可用优惠券，可选择使用
- 了解优惠券使用规则和有效期

### 9.2 积分系统

- 购物可以获得积分
- 积分可以在下单时抵扣现金
- 参与活动可以获得额外积分
- 在"我的"页面查看积分余额和积分明细

## 10. 消息与通知

### 10.1 系统消息

- 订单状态更新通知
- 活动信息推送
- 系统公告

### 10.2 消息设置

1. 进入"我的"页面，点击"设置"
2. 选择"消息通知"
3. 管理各类消息的接收权限

## 11. 售后服务

### 11.1 申请退货/退款

1. 在"我的订单"中找到需要售后的订单
2. 点击"申请售后"
3. 选择售后类型（退货退款、仅退款）
4. 填写退款原因和金额
5. 上传相关凭证
6. 提交申请并等待处理

### 11.2 售后进度查询

- 在"我的"页面，点击"售后服务"
- 查看申请记录和处理进度
- 与客服沟通售后问题
