## 1. 概述

## 2. 账号问题

### 2.1 如何注册账号？

**问题**：如何注册SutWxApp账号？

**解答**：
1. 进入SutWxApp小程序
2. 点击「我的」选项
3. 点击「登录/注册」按钮
4. 选择「微信登录」
5. 点击「允许」按钮，授权微信登录
6. 完成注册，进入个人中心

### 2.2 如何修改密码？

**问题**：如何修改SutWxApp账号密码？

**解答**：
1. 进入个人中心
2. 点击「设置」选项
3. 点击「账号安全」选项
4. 点击「修改密码」选项
5. 输入原密码和新密码
6. 点击「保存」按钮

### 2.3 忘记密码怎么办？

**问题**：忘记SutWxApp账号密码怎么办？

**解答**：
1. 进入SutWxApp小程序
2. 点击「我的」选项
3. 点击「登录/注册」按钮
4. 点击「忘记密码」选项
5. 输入注册手机号
6. 点击「获取验证码」按钮
7. 输入验证码
8. 设置新密码
9. 点击「保存」按钮

### 2.4 如何绑定手机号？

**问题**：如何绑定手机号？

**解答**：
1. 进入个人中心
2. 点击「个人信息」选项
3. 点击「手机号」选项
4. 输入手机号
5. 点击「获取验证码」按钮
6. 输入验证码
7. 点击「绑定」按钮

## 3. 商品问题

### 3.1 如何搜索商品？

**问题**：如何搜索商品？

**解答**：
1. 进入SutWxApp小程序
2. 点击首页顶部的搜索框
3. 输入关键词
4. 点击「搜索」按钮
5. 查看搜索结果

### 3.2 如何筛选商品？

**问题**：如何筛选商品？

**解答**：
1. 进入分类页或搜索结果页
2. 点击「筛选」按钮
3. 选择筛选条件，如价格、销量等
4. 点击「确定」按钮
5. 查看筛选结果

### 3.3 如何排序商品？

**问题**：如何排序商品？

**解答**：
1. 进入分类页或搜索结果页
2. 点击「排序」按钮
3. 选择排序方式，如价格从低到高、销量从高到低等
4. 查看排序结果

### 3.4 如何查看商品评价？

**问题**：如何查看商品评价？

**解答**：
1. 进入商品详情页
2. 滑动屏幕到底部
3. 查看商品评价
4. 点击「查看全部评价」按钮，查看所有评价

## 4. 购物车问题

### 4.1 如何将商品加入购物车？

**问题**：如何将商品加入购物车？

**解答**：

**问题**：如何调整购物车中的商品数量？

**解答**：

**问题**：如何清空购物车？

## 6. 支付问题

### 6.1 支持哪些支付方式？

**问题**：SutWxApp支持哪些支付方式？

### 6.2 支付失败怎么办？

**问题**：支付失败怎么办？

**解答**：
1. 检查网络连接是否正常
2. 检查支付账户余额是否充足
3. 检查支付方式是否可用
4. 重新尝试支付
5. 如仍无法解决，联系客服

## 8. 小程序问题

### 8.1 如何找到SutWxApp小程序？

**问题**：如何找到SutWxApp小程序？

**解答**：
1. 打开微信，点击底部的「发现」选项
2. 点击「小程序」选项
3. 在搜索框中输入「SutWxApp」
4. 点击搜索结果中的「SutWxApp」小程序

### 8.2 小程序无法打开怎么办？

**问题**：小程序无法打开怎么办？

**解答**：
1. 检查网络连接是否正常
2. 检查微信版本是否为6.7.2或更高版本
3. 清理微信缓存
4. 重新打开小程序
5. 如仍无法解决，联系客服

### 8.3 小程序卡顿怎么办？

**问题**：小程序卡顿怎么办？

**解答**：
1. 检查网络连接是否正常
2. 清理微信缓存
3. 关闭其他小程序
4. 重新打开小程序
5. 如仍无法解决，联系客服
