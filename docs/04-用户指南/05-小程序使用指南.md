# SutWxApp 小程序使用指南

## 1. 首次使用

### 1.1 微信授权登录

首次打开 SutWxApp 小程序时，系统会提示您进行微信授权登录。请点击“允许”以完成登录，您无需单独注册账号。

### 1.2 完善个人信息

登录后，建议您前往“我的”页面，点击头像或昵称进入个人资料页，完善您的收货地址、联系方式等信息，以便后续购物和接收服务通知。

## 2. 浏览与搜索商品

### 2.1 首页推荐

小程序首页会根据您的浏览历史和偏好，为您推荐精选商品和热门活动。

### 2.2 分类浏览

点击底部导航栏的“分类”按钮，可以进入商品分类页面，按照不同的商品类别进行浏览。

### 2.3 关键词搜索

在小程序顶部的搜索框中输入您想查找的商品关键词，点击搜索即可查看相关商品。

## 3. 购物流程

### 3.1 商品详情

点击商品图片或名称，即可进入商品详情页，查看商品的详细介绍、图片、价格、评价等信息。

### 3.2 加入购物车

在商品详情页，您可以选择商品的规格、数量，然后点击“加入购物车”按钮，将商品添加到购物车中。

### 3.3 立即购买

如果您想直接购买某件商品，可以在商品详情页点击“立即购买”按钮，直接进入订单确认页面。

### 3.4 购物车结算

点击底部导航栏的“购物车”按钮，进入购物车页面。您可以对购物车中的商品进行数量调整、删除操作，然后点击“去结算”按钮。

### 3.5 订单确认与支付

在订单确认页面，请核对您的收货地址、商品信息、优惠券使用情况等。确认无误后，选择支付方式（目前支持微信支付），点击“提交订单”并完成支付。

## 4. 订单管理

### 4.1 查看订单

进入“我的”页面，点击“我的订单”，即可查看所有订单的列表和详细状态（待付款、待发货、待收货、已完成、已取消等）。

### 4.2 申请退款/售后

在订单详情页，如果订单状态允许，您可以点击“申请退款”或“申请售后”按钮，按照提示提交申请。

### 4.3 确认收货

当您收到商品并确认无误后，请在订单详情页点击“确认收货”按钮。

## 5. 个人中心

### 5.1 个人资料

在“我的”页面，点击头像或昵称进入个人资料页，您可以修改昵称、头像、性别等信息。

### 5.2 收货地址管理

在个人中心，您可以添加、修改和删除您的收货地址。

### 5.3 优惠券

查看您已获得的优惠券，并在购物结算时使用。

### 5.4 帮助与反馈

如果您在使用过程中遇到任何问题或有任何建议，可以通过“我的”页面 -> “设置”-> “帮助与反馈”-> “联系客服”与我们取得联系。

## 6. 设置

### 6.1 语言设置

在“我的”页面 -> “设置”中，您可以切换小程序的显示语言（如中文、英文）。

### 6.2 隐私政策

查阅 SutWxApp 的隐私政策，了解我们如何收集、使用和保护您的个人信息。

### 6.3 关于我们

了解 SutWxApp 的版本信息和相关介绍。
