### 5.1 [接口名称]

#### 5.1.2 请求参数

| 参数名 | 类型 | 必选 | 描述 | 默认值 |
|-------|------|------|------|--------|
| param1 | string | 是 | 参数1的描述 | 无 |
| param2 | number | 否 | 参数2的描述 | 10 |

#### 5.1.3 响应格式

成功响应：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "key1": "value1",
    "key2": "value2"
  }
}
```

失败响应：
```json
{
  "code": 400,
  "message": "参数错误"
}
```

#### 5.1.4 示例代码

```javascript
// 调用示例
async function callApi() {
  try {
    const response = await fetch('https://your-site.com/wp-json/sut-wechat-mini/v1/endpoint', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        param1: 'value1',
        param2: 20
      })
    });
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Error:', error);
  }
}
```
