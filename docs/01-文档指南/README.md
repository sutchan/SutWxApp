# 文档指南

**版本号：1.0.0**
*最后更新时间：2025年11月27日*

本项目包含SutWxApp项目的文档编写规范、模板和贡献指南，确保文档的一致性和高质量。

## 快速导航

- 《00-统一格式规范》：文档格式、命名、结构和内容的统一规范。
- 《01-文档模板》：各类文档的标准模板和内容框架。
- 《02-如何贡献文档》：文档编写、提交修改和维护文档质量的指导。

## 维护规范

- 所有文档必须遵守统一格式规范。
- 新增文档类型需先更新文档模板。
- 文档贡献者应遵循贡献指南。
