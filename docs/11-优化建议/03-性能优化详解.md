# 性能优化详解

## 1. 指标与评估
- 关键路径耗时：首屏、列表加载、下单与支付。
- 稳定性指标：错误率、重试收敛、消息送达。
- 资源占用：CPU/内存/网络与磁盘 IO。

## 2. 客户端优化
- 资源按需与懒加载、骨架屏与占位符。
- 网络优化：请求合并、缓存、降级数据源。
- 组件优化：避免深层嵌套、状态最小化与局部刷新。

## 3. 服务端优化
- 数据库：索引、批量读写、事务边界与锁粒度。
- 缓存：热点缓存、写穿/读穿/击穿防护、TTL 设计。
- 并发：消息队列与异步任务、幂等键与冲突检测。

## 4. 场景案例
- 下单链路：库存校验→价格计算→优惠叠加→扣减一致性。
- 积分抵扣：并发扣减与回退、事务与回滚策略。
- 列表加载：分页、预取与缓存，减少重复请求。

## 5. 监控与回归
- 基准对比：版本前后关键指标对比与回归用例。
- 告警与追踪：异常定位与溯源，持续收敛问题。
