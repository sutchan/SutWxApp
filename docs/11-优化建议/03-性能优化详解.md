# 性能优化详解

## 1. 指标与评估
- 关键路径耗时：渲染、列表加载、下单与支付
- 稳定性指标：错误率、重试成功率、崩溃率
- 资源使用：CPU/内存/网络与磁盘 IO

## 2. 客户端优化
- 资源按需与懒加载、预加载与占位
- 网络优化：请求合并、缓存、降级数据源
- 组合优化：避免频繁渲染、状态最小化与局部更新

## 3. 服务端优化
- 数据库：索引、分库分表、读写分离与缓存
- 缓存：热点缓存、读写穿透/旁路、TTL 策略
- 并发：消息队列与异步任务、锁与事务

## 4. 场景案例
- 下单流程：库存校验→价格计算→异步落单→消息通知
- 分享裂变：并发请求与限流、事务与补偿策略
- 列表加载：分页、预取与缓存，减少重复请求

## 5. 监控与复盘
- 基准对比：新增前后关键指标对比与回归用例
- 告警与排查：异常定位与复盘，持续优化问题
