# SutWxApp 版本发布说明

本文档记录了 SutWxApp 微信小程序项目的版本历史、更新内容、兼容性说明和升级指南。

## 版本规范

项目采用语义化版本规范（SemVer）：MAJOR.MINOR.PATCH

- **MAJOR**：不兼容的破坏性变更
- **MINOR**：向下兼容的新功能
- **PATCH**：向下兼容的问题修复

- iOS：11.0 及以上
- Android：6.0 及以上

## 升级指南

### 从 v1.x.x 升级到 v1.5.0

1. **依赖更新**：
   - 确保微信开发者工具已更新到最新版本
   - 更新项目配置文件中的基础库版本到 2.24.0 或更高

2. **API 变化**：
   - 用户接口新增了多语言相关字段
   - 订单接口返回格式有细微调整
   - 商品接口新增了AR相关参数

3. **配置更新**：
   - 需要在 app.json 中添加新的页面和组件配置
   - 更新小程序权限设置，添加相机权限（用于AR功能）

### 注意事项

- 升级前请确保已备份重要数据
- 建议先在测试环境进行升级测试
- 升级后请全面测试应用功能，特别是新增功能

## 已知问题

- AR功能在部分低端设备上可能存在性能问题
- 离线模式下部分功能可能受限
- 多店铺切换在网络不稳定时可能出现延迟
