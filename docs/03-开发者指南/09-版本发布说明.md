## 1. 概述

### 1.1 文档目的

本文档旨在记录SutWxApp项目的版本更新内容和变更日志，帮助开发者了解版本之间的差异和变更。

### 1.2 版本管理

项目使用语义化版本规范（SemVer）：MAJOR.MINOR.PATCH，其中：
- **MAJOR**：不兼容的破坏性变更
- **MINOR**：向下兼容的新功能
- **PATCH**：向下兼容的问题修复

## 3. 升级指南

### 3.1 从 1.0.0 升级到 1.1.0

1. 更新代码：
   ```bash
   git pull origin main
   ```

2. 更新依赖：
   ```bash
   npm install
   ```

3. 运行数据库迁移：
   ```bash
   npm run migrate
   ```

4. 重启服务：
   ```bash
   npm run server
   ```

### 3.2 从 1.1.0 升级到 1.1.1

1. 更新代码：
   ```bash
   git pull origin main
   ```

2. 更新依赖：
   ```bash
   npm install
   ```

3. 重启服务：
   ```bash
   npm run server
   ```

## 4. 已知问题

### 4.1 版本 1.0.0

- 产品搜索功能在某些情况下可能返回不准确的结果
- 订单状态更新可能存在延迟
- 部分页面加载速度较慢

### 4.2 版本 1.1.0

- 评论功能在某些情况下可能无法正常提交
- 收藏功能可能存在数据不一致问题

### 4.3 版本 1.1.1

- 无已知重大问题

## 5. 兼容性说明

### 5.1 前端兼容性

- 微信小程序：支持微信小程序基础库 2.10.0 及以上版本
- Web 管理后台：支持 Chrome、Firefox、Safari、Edge 等现代浏览器

### 5.2 后端兼容性

- Node.js：支持 Node.js 16.x 及以上版本
- MySQL：支持 MySQL 8.x 及以上版本
- Redis：支持 Redis 6.x 及以上版本
