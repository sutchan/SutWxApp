# OpenSpec 指令

这些指令适用于在本项目中工作的 AI 助手。

当请求满足以下条件时，请始终打开 `@/openspec/AGENTS.md`：
- 提及计划或提案（如 proposal、spec、change、plan 等词语）
- 引入新功能、破坏性变更、架构转变或重大性能/安全工作
- 听起来模糊不清，你需要在编码前查看权威规范

使用 `@/openspec/AGENTS.md` 了解：
- 如何创建和应用变更提案
- 规范格式和约定
- 项目结构和指南

请保持此管理块，以便 'openspec update' 可以刷新指令。
