## 1. 概述

### 1.1 文档目的

本文档旨在介绍用户指南目录的结构和内容，为应用用户提供快速导航和参考。

### 1.2 适用对象

- 普通用户
- 新用户
- 潜在用户
- 客服人员

## 2. 文档结构

用户指南目录包含以下文档：

| 文档名称 | 描述 |
|---------|------|
| 01-快速入门.md | 帮助新用户快速上手应用 |
| 02-用户指南.md | 详细介绍应用的各项功能和使用方法 |
| 03-功能介绍.md | 分模块介绍应用的核心功能 |
| 04-常见问题.md | 解答用户常见问题 |
| 05-小程序使用指南.md | 微信小程序端的使用说明 |
| 06-隐私政策.md | 介绍应用的隐私保护政策 |

## 3. 使用说明

1. 新用户建议首先阅读《01-快速入门.md》，快速了解应用基本操作
2. 然后阅读《02-用户指南.md》，详细学习各项功能
3. 根据需要参考《03-功能介绍.md》，深入了解特定功能模块
4. 遇到问题时查阅《04-常见问题.md》
5. 微信小程序用户请阅读《05-小程序使用指南.md》
6. 了解隐私政策请阅读《06-隐私政策.md》
