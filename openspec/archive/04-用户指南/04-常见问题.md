## 1. 概述

### 1.1 文档目的

本文档回答用户在使用 SutWxApp 过程中遇到的常见问题，帮助用户快速解决问题，提升使用体验。

## 2. 账号与登录
### 2.1 如何登录 SutWxApp？
**问题**：如何登录 SutWxApp？
**回答**：SutWxApp 支持微信授权登录，您可以按照以下步骤登录：
1. 打开 SutWxApp 小程序
2. 首次使用时，会弹出微信授权请求
3. 点击「允许」授权微信登录
4. 授权成功后，自动登录进入首页

### 2.2 忘记密码怎么办？

**问题**：忘记密码怎么办？

**回答**：SutWxApp 使用微信授权登录，不需要设置单独的密码。如果您无法登录，请尝试以下方法：
1. 检查微信是否正常登录
2. 清除微信缓存后重新尝试
3. 重新授权登录

### 2.3 如何修改个人信息？
**问题**：如何修改个人信息？

**回答**：您可以按照以下步骤修改个人信息：
1. 点击底部导航栏的「我的」
2. 点击「个人信息」
3. 可以修改头像、昵称、性别、生日等信息
4. 修改完成后，点击「保存」
### 2.4 如何绑定手机号？

**问题**：如何绑定手机号？
**回答**：您可以按照以下步骤绑定手机号：

1. 点击底部导航栏的「我的」
2. 点击「设置」
3. 点击「绑定手机号」
4. 输入手机号，获取并输入验证码
5. 点击「绑定」完成绑定
### 2.5 如何切换账号？

**问题**：如何切换账号？

**回答**：您可以按照以下步骤切换账号：
1. 点击底部导航栏的「我的」
2. 点击「设置」
3. 点击「退出登录」
4. 退出后，重新授权其他微信账号登录
## 3. 功能使用

### 3.1 如何搜索商品？
**问题**：如何搜索商品？

**回答**：您可以按照以下步骤搜索商品：
1. 点击首页顶部的搜索框
2. 输入关键词，如「手机」
3. 点击「搜索」按钮或键盘上的「搜索」键
4. 浏览搜索结果

### 3.2 如何加入购物车？

**问题**：如何加入购物车？
**回答**：您可以按照以下步骤加入购物车：

1. 进入商品详情页
2. 选择商品规格（如果有）
3. 点击「加入购物车」按钮
4. 可以选择购买数量
5. 点击「确定」，商品成功加入购物车
### 3.3 如何查看购物车？

**问题**：如何查看购物车？
**回答**：您可以按照以下步骤查看购物车：

1. 点击底部导航栏的「购物车」
2. 查看已添加到购物车的商品
3. 可以修改商品数量或删除商品
4. 点击「结算」按钮进行下单
### 3.4 如何查看订单？
**问题**：如何查看订单？

**回答**：您可以按照以下步骤查看订单：
1. 点击底部导航栏的「我的」
2. 点击「我的订单」
3. 可以查看不同状态的订单：待付款、待发货、待收货、待评价、已完成、已取消
4. 点击订单可以查看详情

### 3.5 如何评价商品？
**问题**：如何评价商品？

**回答**：您可以按照以下步骤评价商品：
1. 点击底部导航栏的「我的」
2. 点击「我的订单」
3. 找到待评价的订单，点击「去评价」
4. 对每个商品进行评价，包括评分、评价内容和图片
5. 点击「提交评价」完成评价
### 3.6 如何管理收货地址？

**问题**：如何管理收货地址？
**回答**：您可以按照以下步骤管理收货地址：
1. 点击底部导航栏的「我的」
2. 点击「收货地址」
3. 可以新增、修改、删除或设置默认地址

### 3.7 如何查看积分？
**问题**：如何查看积分？

**回答**：您可以按照以下步骤查看积分：
1. 点击底部导航栏的「我的」
2. 点击「我的积分」
3. 查看当前积分余额和积分记录
### 3.8 如何使用优惠券？

**问题**：如何使用优惠券？
**回答**：您可以按照以下步骤使用优惠券：

1. 进入购物车，选择要结算的商品
2. 点击「结算」按钮
3. 在订单确认页面，选择可用的优惠券
4. 确认订单信息后，完成支付

## 4. 支付与退款
### 4.1 支持哪些支付方式？
**问题**：SutWxApp 支持哪些支付方式？
**回答**：SutWxApp 目前支持微信支付。
### 4.2 如何完成支付？
**问题**：如何完成支付？

**回答**：您可以按照以下步骤完成支付：
1. 选择商品并加入购物车
2. 进入购物车，点击「结算」
3. 确认收货地址和商品信息
4. 选择支付方式（微信支付）
5. 点击「立即支付」
6. 输入支付密码或使用指纹/面容支付
7. 支付成功后，订单生成

### 4.3 支付失败怎么办？

**问题**：支付失败怎么办？

**回答**：如果支付失败，您可以尝试以下方法：

1. 检查网络连接是否正常
2. 检查微信余额或绑定的银行卡余额是否充足
3. 检查微信支付是否被限制
4. 重新尝试支付
5. 如多次尝试仍失败，请联系客服

### 4.4 如何申请退款？

**问题**：如何申请退款？

**回答**：您可以按照以下步骤申请退款：

1. 点击底部导航栏的「我的」
2. 点击「我的订单」
3. 找到需要退款的订单，点击进入详情页
4. 点击「申请退款」
5. 选择退款原因，填写退款说明
6. 点击「提交申请」
7. 等待商家处理，审核通过后退款
### 4.5 退款多久到账？

**问题**：退款多久到账？

**回答**：退款到账时间取决于支付方式：
- 微信支付：一般1-3个工作日到账
- 银行卡支付：一般3-7个工作日到账

具体到账时间可能因银行处理速度而有所不同。
### 4.6 如何查看退款进度？

**问题**：如何查看退款进度？

**回答**：您可以按照以下步骤查看退款进度：

1. 点击底部导航栏的「我的」
2. 点击「我的订单」
3. 找到已申请退款的订单，点击进入详情页
4. 查看退款状态和进度

## 5. 技术问题
### 5.1 小程序加载缓慢怎么办？

**问题**：小程序加载缓慢怎么办？

**回答**：如果小程序加载缓慢，您可以尝试以下方法：
1. 检查网络连接是否正常
2. 清除微信缓存
3. 关闭其他正在运行的小程序
4. 重新进入小程序
5. 更新微信到最新版本
### 5.2 小程序无法打开怎么办？

**问题**：小程序无法打开怎么办？

**回答**：如果小程序无法打开，您可以尝试以下方法：
1. 检查网络连接是否正常
2. 检查微信版本是否为6.7.2或更高
3. 清除微信缓存
4. 重新搜索并进入小程序
5. 重启微信或手机
### 5.3 页面显示异常怎么办？

**问题**：页面显示异常怎么办？

**回答**：如果页面显示异常，您可以尝试以下方法：

1. 下拉刷新页面
2. 重新进入小程序
3. 清除微信缓存
4. 更新微信到最新版本
5. 如问题持续，请联系客服
### 5.4 如何清除小程序缓存？

**问题**：如何清除小程序缓存？
**回答**：您可以按照以下步骤清除小程序缓存：

1. 点击底部导航栏的「我的」
2. 点击「设置」
3. 点击「清除缓存」
4. 在弹出的确认框中点击「确定」
### 5.5 如何更新小程序？

**问题**：如何更新小程序？
**回答**：小程序会自动更新，您也可以按照以下步骤手动更新：
1. 进入微信，点击底部的「发现」
2. 点击「小程序」
3. 找到 SutWxApp，长按图标
4. 点击「删除」，删除小程序
5. 重新搜索并进入 SutWxApp，自动获取最新版本
