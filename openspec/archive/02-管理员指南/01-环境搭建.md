## 1. 概述

### 1.1 文档目的

本文档旨在指导系统管理员完成SutWxApp项目的环境搭建，确保系统能够正常运行。

### 1.2 环境要求

| 环境类型 | 版本要求 |
|---------|---------|
| 操作系统 | Windows 10/11, Ubuntu 20.04+ |
| Node.js | 16.0.0+ |
| npm | 8.0.0+ |
| 微信开发者工具 | 固定版 |
| 数据库 | MySQL 8.0+ |
| Redis | 6.0+ |

## 2. 环境搭建步骤

### 2.1 安装Node.js和npm

1. 访问Node.js官网（https://nodejs.org/）下载并安装Node.js
2. 验证安装是否成功：
   ```bash
   node -v
   npm -v
   ```

### 2.2 安装微信开发者工具

1. 访问微信开发者工具官网（https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html）下载并安装
2. 登录微信开发者工具，选择小程序项目
3. 导入SutWxApp项目

### 2.3 安装数据库

#### 2.3.1 安装MySQL

1. 访问MySQL官网（https://www.mysql.com/）下载并安装MySQL
2. 配置MySQL，创建数据库和用户：
   ```sql
   CREATE DATABASE sutwxapp DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   CREATE USER 'sutwxapp'@'localhost' IDENTIFIED BY 'password';
   GRANT ALL PRIVILEGES ON sutwxapp.* TO 'sutwxapp'@'localhost';
   FLUSH PRIVILEGES;
   ```

#### 2.3.2 安装Redis

1. 访问Redis官网（https://redis.io/）下载并安装Redis
2. 启动Redis服务：
   ```bash
   redis-server
   ```

### 2.4 配置项目

1. 克隆项目仓库：
   ```bash
   git clone https://github.com/sutchan/SutWxApp.git
   ```

2. 安装项目依赖：
   ```bash
   cd SutWxApp
   npm install
   ```

3. 配置环境变量：
   ```bash
   cp .env.example .env
   # 编辑.env文件，配置数据库连接等信息
   ```

### 2.5 启动项目

1. 启动后端服务：
   ```bash
   npm run server
   ```

2. 启动前端开发服务器：
   ```bash
   npm run dev
   ```

3. 在微信开发者工具中预览项目

## 3. 常见问题

### 3.1 Node.js版本不兼容

**问题**：安装依赖时出现版本不兼容错误
**解决方案**：使用nvm管理Node.js版本，切换到推荐版本

### 3.2 数据库连接失败

**问题**：无法连接到MySQL数据库
**解决方案**：
1. 检查MySQL服务是否正常运行
2. 检查数据库连接配置是否正确
3. 检查数据库用户权限是否正确

### 3.3 Redis连接失败

**问题**：无法连接到Redis

**解决方案**：
1. 检查Redis服务是否正常运行
2. 检查Redis连接配置是否正确
