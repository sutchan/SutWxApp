# 苏铁微信小程序上架材料

## 一、应用基本信息

### 1.1 应用名称
- **小程序名称**：苏铁
- **英文名称**：Sut
- **类目**：电商平台 - 花卉植物

### 1.2 应用描述
苏铁是一款面向植物爱好者的微信小程序电商平台，主营精品植物、园艺用品、花卉盆栽等商品。平台提供用户注册登录、商品浏览、购物车管理、订单处理、积分系统、社交分享等核心功能，致力于为用户提供便捷的在线购物体验和专业的植物养护指导。

### 1.3 功能介绍
1. **用户系统**：支持微信授权登录、手机号登录，提供用户信息管理、收货地址管理功能
2. **商品展示**：支持商品搜索、分类浏览、详情查看、产品图片多图展示
3. **购物车**：支持商品添加、数量修改、全选结算、删除商品等功能
4. **订单系统**：支持订单创建、状态跟踪、历史订单查询
5. **积分系统**：支持积分获取、使用、兑换功能，购物可获积分
6. **社交功能**：支持商品分享、评论、点赞功能
