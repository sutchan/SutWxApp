﻿# SutWxApp 微信小程序项目

## 项目概述

SutWxApp是一个基于微信小程序开发框架的应用程序，旨在提供高效、易用的移动应用体验。

## 项目结构

```
SutWxApp/
├── app.js              # 小程序入口文件
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── assets/             # 静态资源
│   └── images/         # 图片资源
├── components/         # 自定义组件
├── images/             # 图片文件
├── pages/              # 页面文件
│   ├── article/        # 文章相关页面
│   ├── user/           # 用户相关页面
│   └── ...             # 其他功能模块页面
├── services/           # 服务层
├── tests/              # 测试文件
└── utils/              # 工具类
```

## 核心功能

- 用户管理：注册、登录、个人信息管理
- 产品展示：产品列表、详情、搜索
- 订单管理：创建订单、订单查询、状态管理
- 社交功能：分享、评论、点赞
- 通知系统：消息推送、通知设置

## 技术栈

- 微信小程序原生开发
- JavaScript (ES6+)
- WXML/WXSS
- 微信API

## 开发环境

- 微信开发者工具
- Node.js
- 微信小程序开发文档

## 部署说明

1. 在微信开发者工具中导入项目
2. 配置app.json中的appid
3. 编译并预览
4. 提交审核
5. 发布上线

## 版本管理

采用语义化版本规范（SemVer）：MAJOR.MINOR.PATCH

- MAJOR：不兼容的破坏性变更
- MINOR：向下兼容的新功能
- PATCH：向下兼容的问题修复
