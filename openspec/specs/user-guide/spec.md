 用户指南

## 目的
本指南旨在向苏铁微信小程序（SutWxApp）的用户提供如何访问、浏览和使用应用核心功能的详细指导，确保用户能够轻松上手并充分利用应用的各项功能。

## 应用介绍

### 产品概述
苏铁微信小程序（SutWxApp）是一款面向用户的移动应用，提供了丰富的产品展示、购买、订单管理等功能，旨在为用户提供便捷、高效的购物体验。

### 核心功能
- **首页展示**：推荐商品、分类导航、搜索功能、轮播图
- **分类浏览**：按分类查看商品、筛选和排序功能
- **购物车**：商品管理、数量调整、价格计算、结算功能
- **订单管理**：订单列表、订单详情、订单状态跟踪、订单操作
- **个人中心**：个人信息管理、收货地址管理、收藏夹、优惠券、设置

## 快速入门

### 访问应用

1. 打开微信并导航到"发现"页面
2. 点击"小程序"入口
3. 在搜索框中搜索"SutWxApp"
4. 在搜索结果中点击"SutWxApp"小程序
5. 您将被引导到应用的首页

### 微信授权登录

1. 首次打开SutWxApp时，您将看到微信授权提示
2. 点击"允许"按钮，授权SutWxApp访问您的基本微信信息
3. 授权成功后，您将自动登录并被重定向到首页

### 浏览商品

1. 在首页，您可以查看推荐商品
2. 上下滑动浏览更多商品
3. 点击商品图片或名称进入商品详情页
4. 在商品详情页，您可以查看详细信息、价格和评价

### 添加到购物车

1. 在商品详情页，选择可用的商品规格（如果适用）
2. 点击"加入购物车"按钮
3. 商品成功添加到购物车时，您将收到确认消息
4. 您可以点击右上角的购物车图标查看购物车中的商品

### 下单购买

1. 在购物车页面，选择您要购买的商品
2. 点击"去结算"按钮
3. 填写您的收货地址（如果尚未填写）
4. 选择您偏好的支付方式
5. 点击"提交订单"按钮
6. 完成支付流程

## 功能使用说明

### 1. 首页

首页是您进入应用后的第一个页面，包含以下核心功能：

- **轮播图**：展示最新活动和推荐商品，点击轮播图项目将导航到相应的活动页面或商品详情页
- **推荐商品**：展示热门商品，支持横向滑动查看更多项目
- **分类导航**：带有图标的主要商品分类，点击时导航到相应的分类页面
- **活动专区**：展示限时活动的区域，点击活动卡片将导航到活动详情页
- **搜索功能**：位于顶部的搜索框，可用于搜索商品

### 2. 分类浏览

分类浏览功能允许您按分类查看商品，包含以下功能：

- **分类层级展示**：左侧显示一级分类，右侧面板显示所选一级分类的二级分类
- **商品筛选**：可按价格、销量和评分等条件筛选商品
- **商品排序**：可按综合评分、销量、价格等排序
- **商品视图切换**：支持在网格视图和列表视图之间切换

### 3. 商品详情

商品详情页展示商品的详细信息，包含以下内容：

- **商品基本信息**：商品图片、名称、价格和库存状态
- **商品规格选择**：可选择商品规格（如颜色、尺寸）
- **商品详情展示**：包含详细的商品描述、参数、材质和使用说明
- **商品评价**：展示用户评价，支持按评价类型（全部、好评、中评、差评）筛选
- **商品操作按钮**：包含加入购物车、立即购买、收藏和分享按钮

### 4. 购物车

购物车功能允许您管理要购买的商品，包含以下功能：

- **购物车商品展示**：显示所有商品，带有勾选、增加数量、减少数量和删除商品的选项
- **购物车选择功能**：支持全选/取消全选功能
- **实时计算总价**：根据所选商品实时计算总价
- **结算功能**：点击"去结算"按钮将导航到订单确认页面

### 5. 订单管理

订单管理功能允许您查看和管理您的订单，包含以下功能：

- **订单列表显示**：显示所有订单，带有按状态筛选选项（全部、待付款、待发货、待收货、待评价、已完成、已取消）
- **订单搜索**：支持订单搜索
- **订单详情查看**：点击订单可查看详细的订单信息，包括商品信息、收货地址、支付方式、订单金额和物流信息（如果已发货）
- **订单操作**：根据订单状态执行相应的操作：待付款订单可立即付款/取消，待发货订单可取消，待收货订单可查看物流/确认收货，待评价订单可评价，已完成/已取消订单可再次购买

### 6. 个人中心

个人中心功能允许您管理个人信息和设置，包含以下功能：

- **个人信息展示**：显示头像、昵称和ID，支持编辑个人信息
- **地址管理**：可添加、编辑、删除地址和设置默认地址
- **收藏夹**：可查看收藏的商品，带有取消收藏和查看详情的选项
- **优惠券**：可查看所有优惠券，支持按类型筛选，显示使用条件和有效期
- **积分管理**：可查看当前积分余额和积分兑换历史
- **客服中心**：可联系在线客服、常见问题和联系方式
- **设置**：可修改密码、绑定手机号、清除缓存、查看关于页面和退出登录

### 7. 支付功能

支付功能提供安全的支付服务，包含以下功能：

- **支付方式选择**：可从支持的支付方式中选择
- **支付流程**：用户完成订单确认后，点击提交订单将导航到支付页面
- **支付成功反馈**：支付成功后，将被重定向回应用，并收到订单确认

### 8. 消息功能

消息功能提供全面的消息通知服务，包含以下功能：

- **系统消息**：显示平台发送的通知，包括活动公告和系统更新
- **订单消息**：显示订单状态变化通知，包括支付提醒、发货通知和配送通知
- **消息设置**：可启用/禁用消息并设置消息类型

## 常见问题

### 1. 如何修改个人信息？

1. 导航到"我的"页面
2. 点击您的头像或昵称
3. 进入个人信息编辑页面
4. 修改相关信息并保存更改

### 2. 如何查看订单状态？

1. 导航到"我的"页面
2. 点击"我的订单"
3. 在订单列表中查看订单状态
4. 点击订单进入订单详情页，可查看更详细的状态信息

### 3. 如何取消订单？

1. 导航到"我的订单"页面
2. 找到要取消的订单
3. 点击"取消订单"按钮
4. 选择原因并确认

### 4. 如何申请退款？

1. 导航到"我的订单"页面
2. 找到需要退款的订单
3. 点击"申请退款"按钮
4. 填写原因和金额
5. 上传凭证
6. 提交申请

### 5. 如何联系客服？

1. 导航到"我的"页面
2. 点击"客服中心"
3. 选择"在线客服"或"电话客服"
4. 发送消息或拨打电话

### 6. 如何修改收货地址？

1. 导航到"我的"页面
2. 点击"地址管理"
3. 选择要修改的地址
4. 点击"编辑"按钮
5. 修改地址信息并保存

### 7. 如何查看物流信息？

1. 导航到"我的订单"页面
2. 找到已发货的订单
3. 点击订单进入订单详情页
4. 点击"查看物流"按钮
5. 查看物流信息

### 8. 如何获取优惠券？

1. 导航到"我的"页面
2. 点击"优惠券"
3. 查看可领取的优惠券
4. 点击"领取"按钮获取优惠券

## 操作指南

### 1. 商品搜索与筛选

1. 在首页或分类页点击顶部搜索框
2. 输入关键词进行搜索
3. 在搜索结果页，可使用筛选条件（如价格、销量、评分）进一步缩小范围
4. 可选择排序方式（如综合排序、销量从高到低、价格从低到高）

### 2. 购物车管理

1. 在商品详情页点击"加入购物车"按钮添加商品
2. 点击右上角购物车图标进入购物车页面
3. 可调整商品数量或删除商品
4. 勾选要购买的商品
5. 点击"去结算"按钮进入订单确认页

### 3. 订单创建与支付

1. 在购物车页面选择商品后点击"去结算"
2. 确认收货地址
3. 选择支付方式
4. 点击"提交订单"按钮
5. 在支付页面完成支付

### 4. 订单评价

1. 导航到"我的订单"页面
2. 找到已完成的订单
3. 点击"去评价"按钮
4. 对每个商品进行评分和评价
5. 点击"提交评价"按钮
