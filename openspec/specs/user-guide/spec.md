# 用户指南规范

## 目的
本规范定义了苏铁微信小程序（SutWxApp）的用户体验和功能，为用户提供如何访问、导航和使用应用核心功能的指导。

## 要求

### 要求：应用访问
系统应为用户提供跨支持平台访问应用的清晰方法。

#### 场景：微信小程序访问
- **当**用户想要通过微信访问应用时
- **则**他们应能够在微信小程序部分搜索"SutWxApp"来查找并打开应用

### 要求：应用界面布局
系统应提供清晰直观的界面布局，包含一致的导航元素。

#### 场景：底部导航栏
- **当**用户与应用交互时
- **则**他们应看到底部导航栏，包含五个主要入口：首页、分类、购物车、订单和我的账户

#### 场景：顶部导航元素
- **当**用户访问任何页面时
- **则**他们应看到顶部导航元素，包括搜索栏、通知中心和带有商品数量的购物车图标

### 要求：首页功能
系统应实现功能丰富的首页，展示推荐商品和活动。

#### 场景：轮播图显示
- **当**用户访问首页时
- **则**他们应看到显示最新活动和推荐商品的轮播图
- **并且**点击轮播图项目应导航到相应的活动页面或商品详情页

#### 场景：推荐商品显示
- **当**用户浏览首页时
- **则**他们应看到推荐热门商品，支持垂直滚动查看更多项目
- **并且**点击商品卡片应导航到商品详情页

#### 场景：分类导航
- **当**用户查看首页时
- **则**他们应看到带有图标的主要商品分类，点击时导航到相应的分类页面

#### 场景：活动专区显示
- **当**用户访问首页时
- **则**他们应看到展示促销活动的区域
- **并且**点击活动卡片应导航到活动详情页

### 要求：分类浏览功能
系统应提供全面的商品分类浏览功能，包含筛选和排序选项。

#### 场景：分类层级显示
- **当**用户访问分类页面时
- **则**他们应看到左侧边栏显示一级分类，右侧面板显示所选一级分类的二级分类
- **并且**点击二级分类应显示相应的商品列表

#### 场景：商品筛选
- **当**用户按分类浏览商品时
- **则**他们应能够按价格、销量和评分筛选商品
- **并且**支持多个筛选条件

#### 场景：商品排序
- **当**用户查看分类商品时
- **则**他们应能够按综合评分、销量、价格和新品排序

#### 场景：商品视图模式
- **当**用户浏览分类商品时
- **则**他们应能够在网格视图和列表视图之间切换

### 要求：商品详情功能
系统应提供详细的商品信息和交互功能。

#### 场景：商品基本信息
- **当**用户访问商品详情页时
- **则**他们应看到商品图片、名称、价格、销量和库存状态
- **并且**支持图片缩放

#### 场景：商品规格
- **当**用户查看商品详情页时
- **则**他们应能够选择商品规格（如颜色、尺寸）
- **并且**根据所选规格实时更新价格和库存

#### 场景：商品详情显示
- **当**用户向下滚动商品详情页时
- **则**他们应看到详细的商品描述、参数、材质和使用说明
- **并且**支持图文混合显示

#### 场景：商品评论
- **当**用户查看商品详情页时
- **则**他们应看到用户评论，支持按评论类型（全部、好评、中评、差评）筛选
- **并且**支持查看评论图片

#### 场景：商品操作按钮
- **当**用户查看商品详情页时
- **则**他们应看到添加到购物车、立即购买、收藏和分享按钮

### 要求：购物车功能
系统应提供全面的购物车管理功能。

#### 场景：购物车商品显示
- **当**用户访问购物车时
- **则**他们应看到所有商品，带有勾选/取消勾选、调整数量和删除商品的选项

#### 场景：购物车选择功能
- **当**用户管理购物车时
- **则**他们应看到全选/取消全选选项
- **并且**实时计算所选商品的总价

#### 场景：购物车价格计算
- **当**用户调整购物车商品时
- **则**他们应看到单价、数量和总价的实时更新
- **并且**支持优惠券和积分抵扣

#### 场景：购物车结账流程
- **当**用户准备结账时
- **则**点击结账按钮应导航到订单确认页面
- **并且**只有已勾选的商品应包含在订单中

### 要求：订单管理功能
系统应提供全面的订单管理功能。

#### 场景：订单列表显示
- **当**用户访问订单列表时
- **则**他们应看到所有订单，带有按状态筛选选项（全部、待付款、待发货、待收货、待评价、已完成、已取消）
- **并且**支持订单搜索

#### 场景：订单详情信息
- **当**用户查看订单详情页时
- **则**他们应看到详细的订单信息，包括商品详情、收货地址、支付方式、订单金额和物流信息（如果已发货）

#### 场景：订单操作
- **当**用户与订单交互时
- **则**他们应能够执行状态适当的操作：待付款订单可立即付款/取消，待发货订单可取消，待收货订单可查看物流/确认收货，待评价订单可评价，已完成/已取消订单可再次购买

### 要求：个人中心功能
系统应提供全面的个人信息和设置管理。

#### 场景：个人信息显示
- **当**用户访问个人中心时
- **则**他们应看到头像、昵称和ID，支持编辑个人信息

#### 场景：地址管理
- **当**用户管理收货地址时
- **则**他们应能够添加、编辑、删除地址和设置默认地址

#### 场景：收藏管理
- **当**用户查看收藏时
- **则**他们应看到收藏的商品，带有取消收藏和查看详情的选项

#### 场景：优惠券管理
- **当**用户查看优惠券时
- **则**他们应看到所有优惠券，支持按类型筛选，并显示使用条件和有效期

#### 场景：积分管理
- **当**用户查看积分时
- **则**他们应看到当前积分余额和积分交易历史
- **并且**如果可用，支持积分兑换

#### 场景：客服访问
- **当**用户需要支持时
- **则**他们应能够访问在线客服、常见问题和联系信息

#### 场景：账户设置
- **当**用户访问设置时
- **则**他们应能够更改密码、绑定手机号、清除缓存、查看关于页面和退出登录

### 要求：支付功能
系统应提供安全的支付功能，支持多种支付方式。

#### 场景：支付方式选择
- **当**用户进行结账时
- **则**他们应能够从支持的支付方式中选择

#### 场景：支付流程
- **当**用户完成订单确认时
- **则**点击提交订单应导航到支付页面
- **并且**支付成功后，他们应被重定向回应用，并收到订单确认

#### 场景：支付记录访问
- **当**用户查看订单时
- **则**他们应能够查看支付记录和详情

### 要求：通知功能
系统应提供全面的消息通知功能。

#### 场景：系统通知
- **当**用户访问通知时
- **则**他们应看到平台发送的通知，包括活动公告和系统更新

#### 场景：订单通知
- **当**用户查看通知时
- **则**他们应看到订单状态变更通知，包括支付提醒、发货通知和配送通知

#### 场景：通知设置
- **当**用户管理通知偏好时
- **则**他们应能够启用/禁用通知并设置通知类型

### 要求：常见问题解决
系统应为解决常见用户问题提供清晰的指导。

#### 场景：地址修改指导
- **当**用户需要修改收货地址时
- **则**他们应通过以下流程获得指导：我的账户 → 地址管理 → 选择地址 → 编辑 → 保存

#### 场景：订单取消指导
- **当**用户需要取消订单时
- **则**他们应通过以下流程获得指导：我的订单 → 查找订单 → 取消订单 → 选择原因 → 确认

#### 场景：退款申请指导
- **当**用户需要申请退款时
- **则**他们应通过以下流程获得指导：我的订单 → 查找订单 → 申请退款 → 填写原因和金额 → 上传凭证 → 提交申请

#### 场景：客服联系指导
- **当**用户需要联系支持时
- **则**他们应通过以下流程获得指导：我的账户 → 客服中心 → 在线客服 → 聊天界面 → 发送消息

### 要求：用户认证

#### 场景：微信授权登录

## 快速入门指南

### 访问应用
1. 打开微信并导航到"发现"页面
2. 点击"小程序"入口
3. 在搜索栏中搜索"SutWxApp"
4. 在搜索结果中点击"SutWxApp"小程序
5. 您将被引导至应用的首页

### 微信授权登录
1. 首次打开SutWxApp时，您将看到微信授权提示
2. 点击"允许"按钮，授予SutWxApp访问您的基本微信信息
3. 授权成功后，您将自动登录并被重定向到首页

### 浏览商品
1. 在首页，您可以查看推荐商品
2. 上下滑动浏览更多商品
3. 点击商品图片或名称进入商品详情页
4. 在商品详情页，您可以查看详细信息、价格和评论

### 添加到购物车
1. 在商品详情页，选择可用的商品规格
2. 点击"加入购物车"按钮
3. 商品成功添加到购物车时，您将收到确认消息
4. 您可以点击右上角的购物车图标查看购物车中的商品

### 下单
1. 在购物车页面，选择您要购买的商品
2. 点击"去结算"按钮
3. 填写您的收货地址（如果尚未填写）
4. 选择您偏好的支付方式
5. 点击"提交订单"按钮
6. 完成支付流程

## 常见问题

### 如何修改个人信息？
1. 导航到"我的"页面
2. 点击您的头像或用户名
3. 进入个人信息编辑页面
4. 修改相关信息并点击"保存"按钮

### 如何查看订单状态？
1. 导航到"我的"页面
2. 点击"我的订单"
3. 在订单列表中查看订单状态
4. 点击订单进入订单详情页，查看更详细的状态信息

### 如何联系客服？
1. 导航到"我的"页面
2. 点击"联系客服"
3. 进入客服聊天界面并发送消息寻求帮助
