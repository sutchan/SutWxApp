 项目概述

## 目的
Sut微信小程序(SutWxApp) 是一个面向普通消费者的电商微信小程序，专注于提供优质的产品和服务，通过创新的积分系统和社交功能，打造一个集购物、社交、娱乐于一体的综合型电商平台。

## 技术栈
- **前端**：微信小程序框架 + JavaScript
- **后端**：Node.js + Express.js
- **数据库**：MySQL + Redis + MongoDB
- **认证**：JWT + RBAC
- **开发工具**：微信开发者工具、VS Code
- **版本控制**：Git + GitHub
- **构建工具**：Webpack
- **测试框架**：Jest + Supertest

## 项目约定

### 代码风格
- 遵循JavaScript/微信小程序代码风格指南
- 使用ESLint进行代码检查
- 类名使用帕斯卡命名法 (PascalCase)
- 方法名和变量名使用小驼峰命名法 (camelCase)
- 私有成员使用下划线前缀 (_privateMember)
- 文件名使用小驼峰命名法 (camelCase)

### 架构模式
- 采用分层架构设计
- 前端使用微信小程序框架
- 后端使用RESTful API设计
- 状态管理使用微信小程序自带状态管理
- 路由管理使用微信小程序路由系统

### 测试策略
- 单元测试使用Jest框架
- 集成测试使用Supertest框架
- API测试使用Postman
- 代码覆盖率要求达到80%
- 提交代码前必须通过所有测试

### Git Workflow
- main分支为稳定分支，用于发布版本
- dev分支为开发分支，用于集成所有功能
- feature/功能名分支用于开发新功能
- fix/问题描述分支用于修复bug
- 提交信息格式：<类型>: <描述>，如 feat: 添加新功能，fix: 修复bug

## 业务背景
- 应用主要面向普通消费者
- 包含多个功能模块和业务流程
- 涉及多种数据交互和业务逻辑
- 需要处理各种用户场景和边缘情况

## 重要约束
- 应用需要支持微信小程序平台
- 必须遵循微信小程序设计规范和开发要求
- 最低支持微信版本：8.0.0+
- 必须保证应用性能和用户体验
- 必须遵守微信小程序审核规则

## 外部依赖

- 第三方物流API
- 云存储服务
- CDN加速服务
